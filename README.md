# Verbs Training – Python Desktop App

## Overview

**Verbs Training** is a Python desktop application designed to help users **learn and memorize verbs** efficiently. It provides an interactive and user-friendly interface for practicing, testing, and tracking progress with common verbs.

This tool is ideal for language learners, students, and anyone looking to improve their vocabulary retention.

---

## Features

- Interactive verb quiz system
- Multiple-choice and fill-in-the-blank exercises
- Randomized verb practice sessions
- Score tracking and progress monitoring
- Simple, clean desktop interface

---

## Technologies Used

- Python 3.x
- Tkinter (GUI development)
- SQLite (optional for progress tracking)
- Random module (for shuffling exercises)

---

## How It Works

1. The app loads a predefined list of verbs (can be extended by the user).  
2. Users are presented with exercises to **recall or select the correct verb form**.  
3. The app provides immediate feedback on correctness.  
4. Users can review scores and track improvement over time.  

---

## Installation

1. Ensure Python 3.x is installed on your system.  
2. Install required dependencies (Tkinter is usually bundled with Python).  
3. Download the source code.  
4. Run the application:

```bash
python verbs_training.py
```

---

## Usage

- Launch the app to start a new practice session.  
- Answer exercises as they appear.  
- Track your performance at the end of each session.  
- Optionally, add custom verbs to expand your practice library.
